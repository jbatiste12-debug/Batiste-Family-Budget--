Monthly Planner

Offline household budget planner with month history, transfers, and local saving.

How to Use This Budget Planner

  1. Enter all income, expenses, bank accounts, and planned transfers.
  2. Use clear manual descriptions — there are no preset dropdowns.
  3. Select or create a month at the top to keep each month separate.
  4. Your data saves automatically on this device.
  5. Use “Export Backup” if you want a file copy of your saved months.
