How to Use This Budget Planner

  1. Swipe left or right between Income, Expenses, Transfers, and Summary.
  2. Planned transfers and overdraft both count as committed money and reduce what is left.
  3. Select or create a month at the top to save each month separately.
  4. Your data saves automatically on this device.
  5. Use Export Backup if you want a copy of all saved months.
